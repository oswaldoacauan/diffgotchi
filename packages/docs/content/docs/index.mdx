Diffgotchi is a TUI for reviewing the diffs an AI agent produces.

Scroll the diff. Drop comments where things look wrong. Mark files done. Hand
the loop back to your agent — the diffgotchi skill knows how to read your
comments and reply. Come back to a green panel.

## Install

```bash
brew install oswaldoacauan/tap/diffgotchi
```

## Run it

```bash
cd your-repo
diffgotchi
```

That's the whole thing. See [Getting Started](/docs/getting-started) for the
flags you'll actually use, and [Usage](/docs/usage) for the keybinds and the
agent loop.
